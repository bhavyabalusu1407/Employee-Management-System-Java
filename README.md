# Employee Management System (Java)

This is a console-based Employee Management System developed using Core Java.

## Features
- Add Employee
- View Employee
- Update Employee
- Delete Employee
- Search Employee
- Sort Employees by Salary

## Technologies Used
- Java
- Object Oriented Programming
- ArrayList (Java Collections)
- Eclipse IDE

## How to Run

1. Open the project in Eclipse
2. Run `Main.java`
3. Use the console menu to manage employee records

## Project Structure

Employee.java – Stores employee details  
EmployeeManager.java – Handles operations like add, delete, update, search  
Main.java – Menu-driven interface for the system
